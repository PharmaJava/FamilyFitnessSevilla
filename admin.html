Necesario para publicar. El token se guarda solo en este navegador.

Sin probar
¿Cómo creo el token? (paso a paso)
  1. Entra en GitHub → Settings → Developer settings → Fine-grained tokens.
  2. En Repository access elige Only select repositories y marca este repositorio.
  3. En Permissions → Repository permissions, pon Contents: Read and write (el mínimo necesario).
  4. Genera el token, cópialo y pégalo arriba. Pulsa Guardar y Probar conexión.

⚠️ Seguridad: el token vive en tu navegador. Usa uno limitado a este repositorio y revócalo si se filtra.
